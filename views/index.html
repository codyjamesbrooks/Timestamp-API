Timestamp Microservice

Coded by Cody

This is a timestamp Microservice API coded by Cody. There are a couple of ways to use this API, all of which return a JSON object with two keys, the UNIX timestamp, and the UTC time.

Example Usage:

Get current Time

simply add 'api' on to the end of the current url

  • Current Time: https://Timestamp-API.codyjamesbrooks.repl.co/api

Using any valid date string

Add any date string onto the end of the url after api

  • Rosa Parks Arrested: https://Timestamp-API.codyjamesbrooks.repl.co/api/1955-12-1
  • Pioneer 10 reaches Jupiter: https://Timestamp-API.codyjamesbrooks.repl.co/api/12 4 1973
  • The Zipper is patented: https://Timestamp-API.codyjamesbrooks.repl.co/api/April 29, 1913

Using a UNIX timestamp

Add any UNIX timestamp onto the end of the url after api

  • Bugs Bunny is debuted: https://Timestamp-API.codyjamesbrooks.repl.co/api/-928774800000
  • John Lennon is shot: https://Timestamp-API.codyjamesbrooks.repl.com/api/345106800000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
