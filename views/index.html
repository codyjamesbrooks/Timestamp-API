Timestamp Microservice

Coded by Cody

This is a timestamp Microservice API coded by Cody. There are a couple of ways to use this API, all very self all of which I have outlined below. Thanks for using the API.

Example Usage:

Get current Time

simply add 'api' on to the end of the url

  • https://Timestamp-API.codyjamesbrooks.repl.co/api

Using any valid date string

Append any date string onto the end of the url after api/

  • https://Timestamp-API.codyjamesbrooks.repl.co/api/1791-12-15
  • https://Timestamp-API.codyjamesbrooks.repl.co/api/12 17 1770
  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
